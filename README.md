# appGestaoFirebase

Aplicativo de gestão desenvolvido com **Firebase**, permitindo controle de dados em tempo real, autenticação de usuários e armazenamento em nuvem.

## 🔹 Funcionalidades

- Autenticação de usuários (login e cadastro)
- Armazenamento de dados em tempo real com Firebase Realtime Database ou Firestore
- Upload de arquivos para Firebase Storage
- Interface simples e intuitiva
- Notificações e atualizações em tempo real

## 💻 Tecnologias utilizadas

- [Firebase](https://firebase.google.com/) (Auth, Firestore/Realtime DB, Storage)
- Linguagem: JavaScript / TypeScript / Dart (dependendo do framework)
- Framework: Flutter / React Native / Web (especifique o usado)
- Git para controle de versão
